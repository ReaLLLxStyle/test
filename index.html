THE DOG

This project aims to showcase information about a dog using a dynamic web layout. It dynamically creates two div elements: one for textual information and another for displaying an image of the dog. The textual div includes a title, paragraph, and a button. The image div displays the dog's image. CSS styles provide a visually appealing layout with animations for interactive elements like buttons and images. Overall, it's a simple yet effective way to present information about a dog on a web page.
